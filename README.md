# Amazon-web-scraper-
Amazon web scraping and and price analysis using python

This project is a Python-based web scraper that extracts product details from Amazon.

## Features
- Extract product title
- Extract price
- Extract ratings
- Store data in CSV

## Tech Stack
- Python
- BeautifulSoup
- Requests
- Pandas

## How to Run
```bash
pip install -r requirements.txt
python scraper.py
